- Use semicolons occasionally.

- Use first person where appropriate ('I found', 'we ran into').
- Include 1-2 mild opinions or subjective judgments ('honestly this surprised me', 'the docs don't make this obvious').
- Add occasional hedging that sounds genuinely uncertain ('I think', 'from what I can tell', 'not 100% sure on this').
- Use casual register shifts: an informal aside, a colloquial synonym ('use' not 'utilize', 'help' not 'facilitate', 'fix' not 'remediate').
