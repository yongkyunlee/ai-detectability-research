## Requirements

The MRP is a **PDF presentation** to be shared via the student's GitHub repo. The GitHub link to the PDF is the deliverable.

### PDF Slide Deck Must Contain:

1. **tl;dr slide** (1 slide) - Motivation, main idea, and results briefly
2. **Literature review** - Related methods
3. **Approach used** - With all relevant links/references
4. **Code used** - Code to achieve the goal
5. **YouTube link** - Pre-recorded demonstration of the built system

All five items above are required.

### Presentation Details:
- Presented by the author during the last onground session
- Duration: approximately 10 minutes
- Topics cover all directions learned through lectures (not limited to those - contact instructor in advance for other topics)
- Refer to example links on the Professor's GitHub as a starting point
